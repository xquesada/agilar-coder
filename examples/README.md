# Examples

Example project configurations generated by the Agilar AI SDLC methodology.

Each directory contains a `CLAUDE.md` showing what a scaffolded project looks like for a specific team mode and tech stack. These are reference examples — use the `scaffold` wizard to generate configuration tailored to your project.

## Examples

### `solo-go/`

**Solo developer + Go.** A single developer building a Go microservice with one AI agent. Kanban workflow, trunk-based development, full working agreements including code review.

Represents the simplest setup: one person, one agent, no ceremony beyond what the working agreements require.

### `team-node/`

**Multi-human team + Node.js/TypeScript.** Three developers building a React + Express dashboard. Full Scrum with two-week sprints, feature branches, mandatory PRs, CI/CD pipeline.

Represents the most structured setup: multiple people, sprint ceremonies, code review via PRs, staging environment before production.
